FFUF Master Cheat Sheet — command reference organized by category

ffuf — master cheat sheet

Click any command to copy  ·  Fuzz Faster U Fool — web fuzzer & directory brute-forcer

Install
Go install (recommended)
go install github.com/ffuf/ffuf/v2@latestcopy
APT (Kali/Debian)
apt install ffufcopy
Brew (macOS)
brew install ffufcopy
Basic Directory Fuzzing
Fuzz path segment
ffuf -u https://target.com/FUZZ -w wordlist.txtcopy
Fuzz with extension
ffuf -u https://target.com/FUZZ.php -w wordlist.txtcopy
Fuzz file with multiple extensions
ffuf -u https://target.com/FUZZ -w wordlist.txt -e .php,.html,.txt,.bakcopy
GET Parameter Fuzzing
Fuzz GET param name
ffuf -u "https://target.com/page?FUZZ=val" -w params.txtcopy
Fuzz GET param value
ffuf -u "https://target.com/page?id=FUZZ" -w values.txtcopy
POST Body Fuzzing
Fuzz POST form field
ffuf -u https://target.com/login -X POST -d "user=admin&pass=FUZZ" -w passwords.txtcopy
Fuzz JSON body
ffuf -u https://target.com/api -X POST -H "Content-Type: application/json" -d '{"user":"FUZZ"}' -w words.txtcopy
Subdomain Fuzzing
Virtual host / subdomain
ffuf -u https://FUZZ.target.com -w subdomains.txtcopy
Vhost via Host header (no DNS)
ffuf -u http://10.0.0.1 -H "Host: FUZZ.target.com" -w subdomains.txtcopy
Header Fuzzing
Fuzz header value
ffuf -u https://target.com/ -H "X-Custom: FUZZ" -w values.txtcopy
Fuzz User-Agent
ffuf -u https://target.com/ -H "User-Agent: FUZZ" -w ua.txtcopy

Multiple Wordlists
Two wordlists (different keywords)
ffuf -u "https://target.com/W1/W2" -w dirs.txt:W1 -w files.txt:W2copy
Two keywords in URL + body
ffuf -u "https://target.com/USER" -d "pass=PASS" -w users.txt:USER -w pass.txt:PASS -mode clusterbombcopy
Pitchfork mode (parallel, same index)
ffuf -w a.txt:A -w b.txt:B -mode pitchfork -u "https://t.com/A?p=B"copy
Inline / Stdin Wordlists
Pipe wordlist from stdin
cat words.txt | ffuf -u https://target.com/FUZZ -w -copy
Generate numbers & pipe
seq 1 1000 | ffuf -u "https://target.com/item?id=FUZZ" -w -copy
Combine two lists (bash)
cat list1.txt list2.txt | sort -u | ffuf -u https://target.com/FUZZ -w -copy
Wordlist Tips
Sort & deduplicate a wordlist
sort -u wordlist.txt -o wordlist.txtcopy
Start from offset (resume / skip)
ffuf -u https://target.com/FUZZ -w wordlist.txt -p 500copy
Match by Status Code
Match only 200
-mc 200copy
Match 200, 301, 302, 403
-mc 200,301,302,403copy
Filter out (hide) 404
-fc 404copy
Filter 404 + 403 + 500
-fc 404,403,500copy
Match / Filter by Size
Match specific response size
-ms 1234copy
Filter out response size (hide false positives)
-fs 1234copy
Filter multiple sizes
-fs 0,1234,4321copy
Match / Filter by Word Count
Match by word count
-mw 100copy
Filter by word count
-fw 100copy
Match / Filter by Lines
Match by line count
-ml 20copy
Filter by line count
-fl 20copy
Match / Filter by Regex
Match body contains "admin"
-mr "admin"copy
Filter body containing error strings
-fr "Not Found|Error|Invalid"copy
Match redirect to specific URL
-mr "Location: /dashboard"copy
Match / Filter by Time
Match responses over 2000ms (SQLi timing)
-mt 2000copy
Filter fast responses (noise)
-ft 50 # hide responses under 50mscopy
Calibration — Auto-filter False Positives recommended
Auto-calibrate (send random strings, learn baseline)
ffuf -u https://target.com/FUZZ -w wordlist.txt -accopy
Custom calibration strategy
-ac -acs advancedcopy
Speed & Threads
Set thread count (default: 40)
-t 100copy
Rate limit (req/sec)
-rate 50copy
Request timeout (seconds)
-timeout 10copy
Delay between requests (ms, range)
-p 0.1 # 100ms delaycopy
Recursion
Enable recursion (follow discovered dirs)
-recursioncopy
Set max recursion depth
-recursion -recursion-depth 3copy
Recursion + custom strategy
-recursion -recursion-strategy greedycopy

Follow redirects

Ignore TLS errors
-kcopy
Use HTTP/2
-http2copy
Set method
-X PUTcopy
Proxy
Route through Burp Suite proxy

SOCKS5 proxy
-x socks5://127.0.0.1:1080copy
Replay matching results to proxy

Base64-encode the fuzz word

HTML-encode

Matchers on Response Time
Blind SQLi time-based detection
ffuf -u "https://target.com/item?id=FUZZ" -w sqli.txt -mt 3000 -t 5copy
Stop Conditions
Stop on first result

Max time (seconds)

Cookie Authentication
Set session cookie
-b "session=abc123xyz"copy
Multiple cookies
-b "session=abc; csrf=def456"copy
Fuzz with auth + fuzz another param
ffuf -u "https://target.com/api/FUZZ" -w endpoints.txt -b "session=abc123"copy
Bearer Token / Authorization Header
Bearer token
-H "Authorization: Bearer eyJhbGci..."copy
Basic auth header
-H "Authorization: Basic dXNlcjpwYXNz"copy
API key header
-H "X-API-Key: FUZZ" -w apikeys.txtcopy
Multiple Headers
Stack multiple -H flags
ffuf -u https://target.com/FUZZ -w dirs.txt -H "Authorization: Bearer TOKEN" -H "X-Forwarded-For: 127.0.0.1" -H "Accept: application/json"copy
Common Security Headers to Fuzz
X-Forwarded-For bypass
-H "X-Forwarded-For: FUZZ" -w ips.txtcopy
Origin header fuzzing (CORS)
-H "Origin: https://FUZZ.target.com" -w origins.txtcopy
Referer fuzzing
-H "Referer: https://FUZZ" -w domains.txtcopy
Output Files
Save results as JSON
-o results.json -of jsoncopy
Save as CSV
-o results.csv -of csvcopy
Save as HTML report
-o results.html -of htmlcopy
Save as MD (markdown)
-o results.md -of mdcopy
All formats at once
-o results -of allcopy
Display Options
Silent mode (results only, no banner)
-scopy
Show full URL in results
-vcopy
Show response body in output
-v -od ./bodies/ # dump response bodies to dircopy
No color (for piping)
-noninteractivecopy
Post-processing Results
Extract found URLs from JSON output
cat results.json | jq '.results[].url'copy
Get only 200s from JSON
jq '.results[] | select(.status==200) | .url' results.jsoncopy
Silent + grep pipeline
ffuf -s -u https://target.com/FUZZ -w words.txt | grep -v "^$"copy

-bcookieCookie string to send
-r—Follow redirects

-msNMatch response size (bytes)
-fsNFilter response size
-mwNMatch word count
-fwNFilter word count
-mlNMatch line count
-flNFilter line count

-mtmsMatch response time ≥ N milliseconds

-ac—Auto-calibrate filters

-encWORD:encEncode keyword (urlencode, base64, html)
