Nmap cheat sheet — organized by category with copyable commands

TCP scan types

nmap -sS <target>SYN stealth scan default
nmap -sT <target>TCP connect scan (no root)
nmap -sA <target>ACK scan — map firewall rules
nmap -sW <target>Window scan
nmap -sM <target>Maimon scan (FIN/ACK)
nmap -sN <target>Null scan — no flags set
nmap -sF <target>FIN scan
nmap -sX <target>Xmas scan (FIN+PSH+URG)

Other protocols

nmap -sU <target>UDP scan slow
nmap -sI zombie <target>Idle/zombie scan (blind)
nmap -sO <target>IP protocol scan
nmap -b ftp <target>FTP bounce scan
nmap --scanflags URGACKPSH <target>Custom TCP flags
Tip: -sS requires root/sudo. Use -sT without privileges — slower but equally effective for open port detection.

Discovery methods

nmap -sn <target>Ping scan — hosts only, no ports
nmap -Pn <target>Skip ping — treat all hosts as up no ping
nmap -PS80,443 <target>TCP SYN ping on specific ports
nmap -PA80 <target>TCP ACK ping
nmap -PU53 <target>UDP ping on port 53
nmap -PE <target>ICMP echo ping
nmap -PP <target>ICMP timestamp ping
nmap -PM <target>ICMP netmask ping
nmap -PO <target>IP protocol ping
nmap --disable-arp-ping <target>Disable ARP discovery

Target specification

nmap 192.168.1.0/24CIDR notation
nmap 192.168.1.1-50Range of IPs
nmap 192.168.1.1,2,5Individual IPs (comma list)
nmap -iL targets.txtRead targets from file
nmap --exclude 192.168.1.5Exclude a host
nmap --excludefile excl.txtExclude hosts from file
nmap -iR 100Scan 100 random internet hosts

Port selection

nmap -p 22 <target>Single port
nmap -p 22,80,443 <target>Multiple ports
nmap -p 1-1024 <target>Port range
nmap -p- <target>All 65535 ports slow
nmap -p U:53,T:80 <target>Mix UDP/TCP ports
nmap -p http,ftp <target>By service name
nmap --top-ports 100 <target>Top 100 most common ports
nmap -F <target>Fast scan — top 100 ports
nmap -r <target>Scan ports sequentially

Port states

openService actively accepting connections
closedAccessible but no service listening
filteredFirewall blocking probe packets
unfilteredAccessible but state unknown (ACK scan)
open|filteredCould not determine state
Pro tip: Combine -p- with -T4 --min-rate 5000 for full port scans that are faster than the default.

Service & version

nmap -sV <target>Version detection
nmap -sV --version-intensity 9 <target>Max intensity probing
nmap -sV --version-light <target>Faster, lighter version scan
nmap -sV --version-all <target>Try every probe

OS detection

nmap -O <target>OS detection root required
nmap -O --osscan-limit <target>Only try if open+closed ports found
nmap -O --osscan-guess <target>Aggressive OS guessing
nmap -A <target>All: OS + version + scripts + traceroute

Traceroute & misc

nmap --traceroute <target>Trace packet path to host
nmap -6 <target>IPv6 scanning
nmap --reason <target>Show reason for port state

NSE basics

nmap -sC <target>Default safe scripts safe
nmap --script=default <target>Same as -sC
nmap --script=banner <target>Grab service banners
nmap --script=vuln <target>Check known vulns intrusive
nmap --script=safe <target>All safe category scripts
nmap --script=auth <target>Auth bypass checks
nmap --script=brute <target>Brute force credentials intrusive
nmap --script=exploit <target>Exploit vulnerabilities dangerous

Common script combos

nmap --script=http-title <target>Web page titles
nmap --script=ssl-cert -p 443 <target>SSL/TLS certificate info
nmap --script=smb-vuln* <target>All SMB vuln checks
nmap --script=ftp-anon -p 21 <target>FTP anonymous login check
nmap --script=dns-brute <target>DNS subdomain brute-force
nmap --script=http-enum <target>Enumerate web directories
--script-args user=admin,pass=pwPass args to scripts
nmap --script-updatedbUpdate NSE script database

Output formats

nmap -oN out.txt <target>Normal (human-readable)
nmap -oX out.xml <target>XML (parseable)
nmap -oG out.gnmap <target>Grepable output
nmap -oA out <target>All three formats at once best practice
nmap -oS out.txt <target>ScRipT kIdDiE (don't use)

Verbosity & debugging

nmap -v <target>Verbose output
nmap -vv <target>Very verbose
nmap -d <target>Debug level 1
nmap -d9 <target>Max debug level
nmap --open <target>Show only open ports
nmap --packet-trace <target>Show all packets sent/received

Timing templates

nmap -T0 <target>Paranoid — IDS evasion, very slow
nmap -T1 <target>Sneaky — IDS evasion
nmap -T2 <target>Polite — reduce bandwidth
nmap -T3 <target>Normal default
nmap -T4 <target>Aggressive — faster CTF/lab

Firewall & IDS evasion

nmap -f <target>Fragment packets (8-byte frags)
nmap --mtu 16 <target>Set custom MTU (must be ×8)
nmap -D RND:5 <target>5 random decoy IPs
nmap -D 10.0.0.1,ME <target>Specific decoys + your IP
nmap -S <spoof-ip> <target>Spoof source IP
nmap -e eth0 <target>Specify network interface
nmap --source-port 53 <target>Use port 53 as source (firewall bypass)
nmap --data-length 25 <target>Append random data to packets
nmap --randomize-hosts <target>Randomize scan order
nmap --proxies socks4://proxy:port <target>Route through proxy chain
Note: Evasion techniques are for authorized testing only. Use on your own systems or with explicit written permission.

Power combos — ready to use

nmap -sC -sV -oA scan 10.10.10.1CTF/HTB initial recon most common
nmap -p- --min-rate 5000 -T4 10.10.10.1Fast full-port scan
nmap -sC -sV -p 22,80,443 -oN targeted.txt 10.10.10.1Deep scan on known open ports
nmap -sU -sS -p U:53,161,T:80,443 10.10.10.1Combined TCP+UDP on key ports
nmap -sn 192.168.1.0/24Discover live hosts on LAN
nmap -A -T4 --script=vuln 10.10.10.1Aggressive vuln scan noisy
nmap --script=smb-vuln-ms17-010 -p 445 10.10.10.1Check for EternalBlue / MS17-010
nmap -sV --script=ssl-enum-ciphers -p 443 10.10.10.1Enumerate TLS ciphers
nmap -Pn -sT -p 80,8080,8443,8888 10.10.10.1Scan common web ports through strict firewall
nmap --script http-methods --script-args http-methods.url-path='/api' -p 80 10.10.10.1Check allowed HTTP methods on API
Recommended workflow: 1) Quick ping sweep → 2) Full port scan → 3) Service detection on open ports → 4) Targeted scripts. Save every scan with -oA so you can grep results later.
